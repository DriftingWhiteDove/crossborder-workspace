# Target Users

## Phase-1 初始用户画像

### 国内跨境小卖家

| 维度 | 描述 |
|------|------|
| 团队规模 | 1-10 人 |
| 平台 | Amazon 为主，部分多平台（AliExpress / Shopee） |
| 年营收 | 50-500 万 RMB |
| 技术能力 | 会用 Excel，不会写代码 |
| 工具现状 | 多个网页后台切换 + 手动记账 |

### 核心痛点

- 不会计算真实利润（平台费用、头程、关税、退货损耗算不清）
- 竞品价格变动靠人工盯
- Listing 优化凭感觉，没有数据支撑
- 选品靠刷 Best Seller，效率低
- 多平台数据割裂，无法统一看全局

### 用户故事

> "我就想知道这个产品到底赚不赚钱。"
> —— 典型小卖家

## 后续阶段扩展
