# 系统架构

## 架构总览

```
┌─────────────────────────────────────────────────────┐
│                  Frontend (Vue 3)                    │
│  ┌─────────┐ ┌──────────┐ ┌──────────────────────┐  │
│  │ 看板    │ │ 选品分析 │ │ 运营管理             │  │
│  │ Dashboard│ │ Product  │ │ Operations           │  │
│  └─────────┘ └──────────┘ └──────────────────────┘  │
├─────────────────────────────────────────────────────┤
│                  API Gateway (Nginx)                 │
├─────────────────────────────────────────────────────┤
│                  Backend (FastAPI)                   │
│  ┌──────────┐ ┌──────────┐ ┌────────────────────┐  │
│  │ Auth     │ │ Product  │ │ AI Service         │  │
│  │ Service  │ │ Service  │ │ (DeepSeek/Claude)  │  │
│  └──────────┘ └──────────┘ └────────────────────┘  │
├─────────────────────────────────────────────────────┤
│                  Data Layer                          │
│  ┌──────────────┐ ┌──────────┐ ┌────────────────┐  │
│  │ PostgreSQL   │ │ Redis    │ │ 数据抓取调度    │  │
│  │ (业务数据)   │ │ (缓存)   │ │ (Celery + Beat) │  │
│  └──────────────┘ └──────────┘ └────────────────┘  │
└─────────────────────────────────────────────────────┘
```

## 核心服务

### Auth Service
用户注册、登录、OAuth 集成与 JWT 鉴权。

### Product Service
商品数据聚合、搜索、筛选与详情展示，对接各平台 API。

### AI Service
基于 DeepSeek/Claude API 的文案生成、市场分析、智能推荐。

### Data Pipeline
定时抓取任务管理、数据清洗与指标计算，基于 Celery 异步处理。
